To-Do List App

The To-Do List App is a lightweight, responsive application that helps users manage tasks with ease. You can add, edit, delete, and filter tasks by date or keyword. Each task can have an optional image attached, and completed tasks are dynamically separated from pending ones.

Key Features:

  • Light & Dark Modes: Toggle between themes for a personalized experience.
  • Task Management: Add, edit, and delete tasks. Attach images for more context.
  • Filtering & Search: Filter tasks by date or search keywords.
  • Completed List: Seamlessly move tasks between active and completed sections.

This project demonstrates my ability to build reactive, stateful apps using modern frontend tools and user-centric design.

Technologies Used: HTML, CSS, React JavaScript, Ant Design
