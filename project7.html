Custom Experience Builder Widgets

Custom widget development for ArcGIS Experience Builder applications, focused on reusable GIS workflows such as filtering, map interaction, data display, and service-driven UI behavior.

Key Contributions:

  • Widget architecture: Built modular frontend logic using React, TypeScript, and Experience Builder conventions.
  • Map integration: Connected widgets with ArcGIS data sources, Web Maps, and Feature Layer workflows.
  • Interactive filtering: Supported user-driven filtering and query behavior for dashboard and map-centric applications.
  • Maintainable UI: Organized components and styles for easier reuse across multiple GIS applications.

This project area aligns with Esri's Experience Builder Developer Edition model, where custom widgets are developed with TypeScript, React, JSX, and the Jimu framework.

Technologies Used: ArcGIS Experience Builder Developer Edition, Jimu, React, TypeScript, JavaScript, HTML5, CSS
